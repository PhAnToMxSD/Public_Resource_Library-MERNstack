# Public_Resource_Library-MERNstack
This is a public resource library where people can upload links/documents as resources, to help the community grow and learn from each other. 

This website has the following features :- 
1. People can upload resources in the form of links or documents, and select the categories of their uploaded resources. 
2. People can keep a track of the resources that they have uploaded to the E-library. 
3. People can search for specific resources and have access to them. They can search for resources in two ways - {For specific resource - by entering the name} or {By selecting a category of resources}
